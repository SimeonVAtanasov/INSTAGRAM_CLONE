# 300gram

### Final Project for IT Talents Training Camp Season 12

# Overview

Single Page Application, copy of the original Instagram (https://instagram.com) . 


#  Technologies :

 * Javascript 
 * React
 * Redux
 * Redux-Thunk
 * CSS  & SCSS modules
 * Material UI
 * Firebase
 
 
# Features

* Register/ login / logout 
* Uploading posts & stories
* Like & unlike posts
* Save posts
* Comment posts
* Follow & unfollow
* Chat room
* Notifications
